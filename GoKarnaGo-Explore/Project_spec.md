# Project Specification: GoKarnaGo-Explore (Mobile App)
**Agency:** TheBrainCord (TBC)
**Type:** Taxi Booking & Tour Operator App (Flutter)

## 1. Business Logic
- **Goal:** Connect tourists (specifically foreigners) with local drivers in Gokarna.
- **Unique Selling Point:** Direct connection, negotiation enabled, minimal commission model.
- **Key Routes:** Airport pickups/drops from Goa (GOI/GOX), Hubli (HBX), and Mangalore (IXE) to Gokarna.
- **Services:** Round trips, 1/2 day sightseeing, Temple tours.

## 2. Technical Stack
- **Frontend:** Flutter (Mobile - iOS/Android).
- **Backend/Integration:** Firebase (for initial MVP) or Node.js.
- **DevOps:** Docker (for containerizing the build environment), GitHub Actions (CI/CD).
- **AI Feature:** "Linguist-Link" - A real-time chat translation tool to help foreign tourists converse with local drivers (English <-> Kannada/Hindi/Konkani).

## 3. Core Features
- **Driver Direct:** Users see a list of drivers, their specific cars, and ratings. Can "Request Quote" or "Book Direct".
- **Translation Chat:** In-app messaging interface where text is auto-translated.
- **Tour Packages:** Pre-defined route cards for "Gokarna Temple Tour" or "Om Beach Sunset".
